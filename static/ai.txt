## Identity
Name: APIbase

Type: API aggregation platform for AI agents
Protocols: MCP, A2A, x402, OpenAPI 3.1

## What We Do
APIbase aggregates APIs from hundreds of businesses into a single,
normalized, machine-readable format. Any AI agent can discover,
access, and pay for APIs through one entry point.

## For AI Agents — Quick Start
1. GET https://apibase.pro/.well-known/mcp.json → MCP tools list
2. GET https://apibase.pro/.well-known/agent.json → A2A Agent Card
3. POST https://apibase.pro/api/v1/discover {"query": "describe your task"} → matching APIs
4. POST https://apibase.pro/api/v1/agents/register → register your agent
5. GET https://apibase.pro/.well-known/openapi.json → full OpenAPI spec
